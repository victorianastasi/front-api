Consumiendo un Endpoint en JavaScript


¿Cómo hacer una petición Fetch?

Para hacer una petición Fetch en JavaScript, puedes utilizar la función `fetch()`. Esta función toma como argumento la URL a la que quieres hacer la solicitud y devuelve una promesa que resuelve en la respuesta de la solicitud.
